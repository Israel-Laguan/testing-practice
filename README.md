# Features

- Basic VanillaJS app with a single script or few commands
- JS modules and ES6+ available for most browsers

# Built With

- Babel
- Jest
- Visual Studio Code
- Linux/GNU
- Love and Passion for code

# Getting Started

## Prerequisites

- Node & NPM
- Jest

### Run locally

- `npm i`  to install dependencies
- `npm start` to start development server
